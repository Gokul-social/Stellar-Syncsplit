# Google Form Setup Guide — SYNC_SPLIT User Feedback

## Step-by-Step: Create the Google Form

1. Go to [forms.google.com](https://forms.google.com) → **Blank form**
2. Title: **SYNC_SPLIT — Beta User Feedback**
3. Description: *Thank you for testing SYNC_SPLIT! This takes 2 minutes.*

---

### Question 1 — Short Answer
**Your Full Name** *(Required)*

**Email Address** *(Required)*

*(Required)*

**Rate your overall experience with SYNC_SPLIT**

- 5 = Love it, would recommend

*(Required)*

---

## After Collecting Responses

1. In Google Forms → **Responses** tab → click the **Google Sheets icon** to create a linked sheet
2. In Google Sheets → **File → Download → Microsoft Excel (.xlsx)**
3. Save as `docs/user_feedback.xlsx` in your project
4. Update the README link to point to this file
